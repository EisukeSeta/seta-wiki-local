## 📝 作成コンセプト

### AIの透明性と制御の可視化
本プロジェクトの核心は、**「AI（Antigravity）が今どこにアクセスでき、何を操作しているのか」をユーザーが直感的に把握できる状態にする**ことです。AIエージェントの作業範囲はブラックボックス化しやすいため、これをWebベースのUIで常に公開・可視化することで、安全かつ確実なペアプログラミング環境を提供します。

### グラスモフィズムによるモダンなUX
「未来のAIコントロールパネル」を感じさせるデザインを目指し、グラスモフィズム（透過的な背景とぼかし）を採用しました。単なる管理ツールではなく、使っていて心地よい、開発者のモチベーションを高めるインフラストラクチャとしての側面を持たせています。

---

## ✨ 主要機能

### 🔍 リアルタイム・インテリジェント検索
- **動作**: 1文字ごとのインクリメンタルサーチ。
- **特徴**: 階層構造を維持したまま絞り込むため、ファイルが「どのプロジェクトのどの階層」にあるかを失わずに検索可能です。
- **視認性**: ヒット箇所の自動ハイライト。

### 🖼️ マルチ形式プレビュー
ブラウザ上で以下のファイルを即座に確認できます。
- **画像**: 主要な画像形式（SVG含む）の自動スケーリング表示。
- **ドキュメント**: `.md` ファイルをリッチなHTMLに変換してレンダリング。
- **コード**: シンタックスハイライト風の等幅フォントを使用し、エスケープ処理されたコードを表示。
- **特殊ファイル**: 拡張子のない `LICENSE` や `NOTICE` 等のテキスト判定。

### 🔄 ダイレクト・シンクロナイザー
- フロントエンドからバックエンドの Node.js スクリプトを叩き、ローカルのファイルシステムを再スキャン。
- 常に最新の「AIの視界」をUIに反映します。

## 🚀 使用方法・デプロイ

### 起動手順
1. リポジトリのルートでサーバーを起動します。
   ```bash
   node viewer/server.js
   ```
2. ブラウザで下記URLにアクセスします。
   ```
   http://localhost:8000/viewer/index.html
   ```

### 構成ファイル
| ファイル | 役割 |
| :--- | :--- |
| `index.html` | メインUI、検索・プレビューエンジン |
| `server.js` | 静的ファイル提供・APIエンドポイント |
| `update_files.js` | ファイルシステムの高速スキャン・メタデータ生成 |

##  Telescope 今後の展開とロードマップ

### 1. マルチ・ワークスペースの統合
現在は単一のプロジェクトルートを対象にしていますが、今後は複数のワークスペースをトップレベルで切り替え、あるいは横断して検索できる機能を追加予定です。

### 2. 権限管理の可視化（Safety Monitor）
Antigravityが「読み取りのみ」「書き込み可能」「アクセス拒否」のいずれの状態であるかを、ファイルごとにアイコンやバッジで表示し、AIの行動制限を視覚的に監視できるようにします。

### 3. バイナリ・詳細アナライザー
実行ファイル（.exe）や巨大なデータの構造を、展開せずにプレビュー上で解析する機能を追加し、より深いレベルでのプロジェクト理解を支援します。

### 4. Git 連携の強化
プレビュー画面から直接 GitHub の該当行を開いたり、コミット履歴を確認したりする機能を統合し、GitHub とローカルの架け橋としての機能を強化します。
